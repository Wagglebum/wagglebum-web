Getting started

Three lines. That's the whole API.

Drop the package in, create a schema, call WagSave.Save(). That's it. Cloud sync is opt-in from the inspector.

Read the full docs → 
// schema.cs
public class Run : WagSaveData {
  public int level;
  public float timePlayed;
  public List<Item> inventory;
}

// anywhere in your game:
WagSave.Save("run-1", myRun);
var loaded = WagSave.Load<Run>("run-1");
